OpenBacklog is a self-hosted project management tool for solo developers who want structure without enterprise overhead. Manage tasks, plan strategy, and connect your own AI via MCP.

## Quick Start

```bash
git clone https://github.com/openbacklog/openbacklog.git
cd openbacklog
docker-compose --env-file .env.development up
```

Open http://localhost:8000 to get started.

See the [Quickstart Guide](https://docs.openbacklog.ai/quickstart) for detailed setup instructions and [Claude Code Setup](https://docs.openbacklog.ai/mcp-integration/claude-code) to connect your AI.
