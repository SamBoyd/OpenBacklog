## [0.1.0] - 2025-01-07

### Added

#### Strategic Planning
- Product vision definition and management
- Strategic pillars with strategy and anti-strategy statements
- Product outcomes with measurable goals and pillar linkage
- Roadmap themes with prioritization and outcome alignment

#### Narrative Framework
- Hero personas to represent target users
- Villain definitions to capture problems and obstacles
- Conflict tracking between heroes and villains
- Story arc visualization for product narrative

#### Work Management
- Strategic initiatives with full narrative connections
- Tasks with checklists and initiative context
- Status workflow (To Do, In Progress, Blocked, Done, Archived)
- Task types (Coding, Testing, Documentation, Design)

#### MCP Integration
- Full MCP server for AI assistant integration
- Strategic context summary for AI decision-making
- Entity creation and management via MCP tools
- Framework guidance for entity creation

#### Self-Hosted Infrastructure
- Docker Compose deployment
- Multi-cluster development support
- PostgreSQL with PostgREST API layer
- Auth0 integration for authentication

#### Web Interface
- React-based single-page application
- Story Bible for strategic foundation
- Roadmap view with theme prioritization
- Kanban board for work management
