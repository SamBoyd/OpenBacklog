Frequently Asked Questions

What's the basic workflow in OpenBacklog?

Create Initiatives (larger goals) → Break them into Tasks → Use AI assistance to refine and manage your work. The platform helps organize your development workflow with AI-powered insights.

How do I set up my OpenAI API key?

Go to your Account Settings, navigate to the "OpenBacklog Token Management" section, and add your OpenAI API key. This enables AI-assisted features throughout the platform.

How do I connect GitHub integration?

GitHub initegration is currently in developement and hopefully will be released soon. It'll allow you to reference files in the ticket descriptions and link issues, PRs etc.

How do workspaces work?

Workspaces help organize different projects or contexts. Currently user's are restricted to a single workspace, though I aim to change this as soon as possible.

Is OpenBacklog free to use?

OpenBacklog is open source and free to self-host. Our hosted version has usage-based pricing for AI features. Check your usage and billing in Account Settings.
